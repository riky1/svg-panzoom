Pan & Zoom for inline SVG

A framework-agnostic JavaScript library that adds drag to pan, scroll to zoom, and pinch zoom to any inline SVG element in the DOM.

riky1/svg-panzoom
  • 🖱 Drag to pan
  • ⚙️ Scroll wheel to zoom
  • ✌️ Pinch to zoom on touch devices
  • 🎯 Use the toolbar buttons to control zoom & position
